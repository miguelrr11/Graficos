Hop-in-One es un videojuego 3D que transforma el minigolf clásico en una experiencia de plataformas y agilidad contrarreloj. El jugador debe avanzar por archipiélagos flotantes generados proceduralmente y encestar la bola antes de que el tiempo se agote. Cada hoyo superado añade 30 segundos al contador, por lo que el objetivo es completar la mayor cantidad posible de niveles cada vez más complejos.
